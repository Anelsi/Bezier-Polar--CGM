PolarBezier (de Casteljau) — User Guide

Какво е това: JavaFX приложение за Bézier крива от произволен ред (n = брой контролни точки − 1), оценяване и визуализация с алгоритъма на de Casteljau. Има избор на точки от екрана, anti-aliasing визуално и режим “step forward”.

1) Изисквания

  • JDK 17 (инсталиран на компютъра)
  • Проектът включва Gradle Wrapper (gradlew/gradlew.bat), не е нужно да се инсталира Gradle отделно

2) Стартиране (най-лесно) — от Windows с 1 клик

В корена на проекта има файл run.bat. Стартирайте го с двоен клик.

3) Стартиране през Terminal (Windows)

  1. Разархивирай проекта (ако е .zip)
  2. Отвори Command Prompt в папката на проекта
  3. Изпълни:
gradlew.bat run

4) Стартиране през IntelliJ

  1. Open проекта
  2. Gradle tool window → Tasks → application → run

5) Как се използва приложението

  • Ляв клик: добавя контролна точка
  • Drag: мести контролна точка
  • Десен клик върху точка: трие точка
  • Clear: изчиства всички контролни точки.
  • Degree (n): показва реда на кривата (n = points − 1)
  • t slider: избира параметър t и показва точката B(t)
  • Show de Casteljau levels: показва междинните нива на алгоритъма
  • Example 1/2/3: зареждат готови контролни точки за бърза демонстрация на кривата.
  • Play: рисува “стъпка напред”

6) Файлова структура

src/main/java/org/example/Main.java  (основният код)
build.gradle.kts                     (Gradle настройки + JavaFX)
gradlew / gradlew.bat                (Gradle wrapper)
gradle/wrapper/                      (wrapper jar + properties)
run.bat                              (1-клик стартиране)
docs/index.html                      (този user guide)
